Windows系統

Windows的用戶請直接從Hugo GitHub下載安裝檔解壓後直接安裝即可。
Release Version 0.55.5 檔案連結

macOS系統

macOS則必需透過Homebrew這個套件管理軟體來進行安裝，所以首先要安裝Homebrew，當然若您已安裝過此套件管理工具，則可略過此步驟。
